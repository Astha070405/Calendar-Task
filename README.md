📅 Interactive Calendar Component

🚀 Overview

This project is a modern, interactive calendar component built using Next.js and React, inspired by a physical wall calendar design. It combines clean UI with functional features like date range selection, notes, and dynamic visuals.

---

✨ Features

📅 Interactive Calendar

Navigate between months
Clean grid-based layout

🎯 Date Range Selection

Select start and end dates
Highlighted range with visual feedback

📝 Notes Section

Add notes for planning
Data persists using localStorage

🎨 Dynamic Hero Section

Image changes with month
Theme adapts visually

📊 Insights Panel

Total selected days
Weekend count

📈 Activity Heapmap

Visual representation of selected range

📱 Fully Responsive

Works seamlessly on mobile and desktop

🛠 Tech Stack

Next.js (App Router)
React
Tailwind CSS
Zustand (State Management)
date-fns

📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Astha070405/Calendar-Task.git
cd Calendar-Task
npm install
npm run dev
```
